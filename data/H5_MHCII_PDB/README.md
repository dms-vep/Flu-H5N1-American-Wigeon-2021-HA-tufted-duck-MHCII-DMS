# MHC-II–H5 HA Structure

This folder contains the [PDB file](H5_MHCII_36XU.pdb) for the structure of MHC-II bound to H5 influenza HA.

The structure has been submitted to the Protein Data Bank under accession **36XU**.


[constructs.csv](constructs.csv) contains HA, tufted duck MHC-II and NA protein sequences used for recombinant protein expression for structural analyses presented in associated manuscript.
