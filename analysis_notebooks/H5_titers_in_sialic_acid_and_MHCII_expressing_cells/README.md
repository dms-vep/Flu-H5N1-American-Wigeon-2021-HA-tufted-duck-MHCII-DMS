# H5 Titers in Sialic Acid and MHCII Expressing Cells

This notebook plots American Wigeon H5 pseudovirus titers on cells that have or lack sialic acid and express tufted duck MHCII.

## Project Structure

```
H5_titers_in_sialic_acid_and_MHCII_expressing_cells/
├── virus_titers.ipynb          # Main analysis notebook
├── virus_titers.html           # Output visualization
├── data/
│   └── virus_titer_RLU.csv    # Input RLU titer data
└── README.md                   # This documentation
```

## Input Files

- **`data/virus_titer_RLU.csv`** (3KB): Contains RLU measurements for virus titers

## Output Files

- **`virus_titers.html`** (3KB): Interactive plot generated by the notebook

## Usage

This notebook needs to be run manually.

```bash
jupyter notebook virus_titers.ipynb
```
