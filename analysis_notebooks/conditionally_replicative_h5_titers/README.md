# Conditionally Replicative H5 Virus Titers

This notebook plots titers of conditionally replicative H5 viruses on different cell lines. These H5 viruses lack PBSC in HA, are made in WSN background and have PB1 coding region replaced by GFP and therefore they can only replicate on cells that express PB1 gene.

## Project Structure

```
conditionally_replicative_h5_titers/
├── plot_titers.ipynb                    # Main analysis notebook
├── conditional_flu_virus_titers.html    # Output visualization
├── data/
│   └── virus_titers.csv                # Input titer data
└── README.md                           # This documentation
```

## Input Files

- **`data/virus_titers.csv`** (3.5KB): Contains virus titer measurements

## Output Files

- **`conditional_flu_virus_titers.html`** (12KB): Interactive plot generated by the notebook

## Usage

This notebook needs to be run manually.

```bash
jupyter notebook plot_titers.ipynb
```
